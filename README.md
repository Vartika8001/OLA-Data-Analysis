# 🚖 OLA Data Analysis Project | Excel + SQL + Power BI

This project is an **end-to-end Data Analysis project** based on **OLA ride booking data**, where **SQL** was used for data extraction and analysis, and **Power BI** was used to build interactive dashboards for business insights.

The project focuses on analyzing **ride bookings, cancellations, revenue, customer ratings, and vehicle performance**, helping derive actionable insights for operational improvements and better customer experience.

---

## 📌 Project Objectives

The main objectives of this project were:

- Analyze ride booking trends
- Monitor booking success and cancellations
- Evaluate customer and driver ratings
- Measure revenue performance
- Understand vehicle type utilization
- Build interactive dashboards for decision-making

---

## 🛠 Tools & Technologies Used

- **SQL (MySQL)** – Data querying and analysis  
- **Power BI** – Dashboard creation and visualization  
- **Excel / CSV** – Dataset handling  
- **GitHub** – Version control and project hosting

## 📊 SQL Analysis Performed

The following business queries were solved using SQL:

### 1. Successful Bookings
Retrieved all rides completed successfully.

### 2. Average Ride Distance by Vehicle Type
Analyzed the average trip distance for each vehicle category.

### 3. Customer Cancellation Analysis
Calculated the total number of rides cancelled by customers.

### 4. Top Customers
Identified the top 5 customers based on booking frequency.

### 5. Driver Cancellation Reasons
Analyzed rides cancelled by drivers due to personal or vehicle issues.

### 6. Driver Rating Analysis
Found the maximum and minimum driver ratings for Prime Sedan rides.

### 7. Payment Method Analysis
Retrieved rides paid via UPI.

### 8. Customer Ratings by Vehicle Type
Measured average customer satisfaction across vehicle categories.

### 9. Revenue Analysis
Calculated total booking value for successful rides.

### 10. Incomplete Ride Analysis
Listed incomplete rides along with their reasons.

## 📈 Power BI Dashboard Insights

Interactive dashboards were created to visualize:

### 1. Ride Volume Over Time

### 4. Revenue Insights

## 📌 Key Business Insights

- Ride success rate forms the majority of total bookings, indicating stable operations.
- Customer cancellations reveal operational delays and booking behavior trends.
- Certain vehicle types generate more ride distance and revenue, helping optimize fleet allocation.
- Payment method analysis helps identify customer payment preferences.
- Ratings trends highlight customer satisfaction patterns.
